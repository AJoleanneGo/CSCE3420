Coding a Library System

Background:

A local public library needs a digital system to keep track of its catalog as it has been using manual spreadsheet-style bookkeeping.

Students were tasked to:

  • Code a simple structure for computer-based bookkeeping in C++.
  • Create a report with file relationships and coding API

Achievements:

Full Marks on the assignment


Results:

The code I created is based on a simple API document I created with all of the commands and classes, and how they interact with each other.

My program:

  • Creates a database of library items, patrons, and loans stored in arrays that can be accessed and searched by library employees
  • Includes check-in, check-out, and renewal functions for borrowed library items
  • Keeps track of due dates and updates a loan amount per patron
